Headphones Landing Page
A modern, fully responsive single-page website for a headphones/audio company. Built with semantic HTML5 and CSS3, featuring a hero section with navigation, services showcase, statistics display, contact form, and footer.
Project Description
This project is a complete landing page consisting of five main sections designed to create an engaging user experience for visitors. The page features smooth animations, responsive design, and a mobile-first approach with a CSS-only hamburger menu.

Sections Overview
The landing page is structured in the following order:
1. Hero Section
The landing page opens with a captivating hero section featuring:

Responsive Navigation Bar: Logo on the left, menu links on the right
CSS-Only Hamburger Menu: Transforms into a mobile menu on small screens (no JavaScript required)
Hero Content: Large headline, subtitle, and call-to-action button
Background: Dark gradient overlay on background image
Description Text: Additional context below the CTA button

Key Features:

Fixed navigation with pentagon logo icon
Menu items: "what we do", "our results", "contact us"
Slide-in mobile menu using pure CSS checkbox technique
Fully responsive layout

2. What We Do Section
Showcases the company's services with:

Centered Heading: "What we do..."
Description Paragraph: Lorem ipsum text explaining services
Four Service Cards: Each featuring:

Font Awesome icon (sound wave, video, music note, ear/hearing)
Service title
Brief description


Clean Layout: Icons and text centered, cards evenly spaced

Design Details:

Light gray background (#f8f8f8)
Coral-colored icons matching brand theme
Responsive grid that stacks on mobile

3. Results Section
Eye-catching statistics display featuring:

Dark Blue Gradient Background: Creates dramatic contrast
Decorative Elements: Circular pattern overlays for visual interest
Four Pentagon-Shaped Cards: Each displaying:

Large percentage statistic (+2%)
Descriptive label text


Hover Effects: Cards lift slightly on hover

Technical Details:

CSS clip-path for pentagon shapes
Gradient backgrounds on cards
Fully responsive card layout
Decorative SVG patterns in background

4. Contact Form Section
Professional contact form with:

Centered Form Container: White card on light background
Three Input Fields:

Name (text input)
Email (email input)
Your message (textarea)


Minimalist Design: Underlined inputs with no borders
Call-to-Action Button: Coral-colored submit button

UX Features:

Focus states on inputs
Clean, uncluttered design
Proper form labels
Responsive padding and spacing

5. Footer Section
Professional footer completing the page:

Logo: Pentagon icon with "HEADPHONES" text
Social Media Links: Three circular icons (Facebook, Twitter, Instagram)
Copyright Information: Centered at bottom
Dark Navy Background: Consistent with brand colors

Layout:

Two-column layout (logo left, social right)
Centered copyright text
Hover effects on social icons
Fully responsive with centered mobile layout

Features

✅ 100% Responsive: Works seamlessly on all device sizes
✅ CSS-Only Mobile Menu: No JavaScript required for navigation
✅ Modern Design: Clean, contemporary aesthetic
